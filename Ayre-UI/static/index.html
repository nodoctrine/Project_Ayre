Skills

▾

Reusable workflows Ayre can follow on demand. Built-in skills are always available. Custom skills are saved locally to your Ayre and work across all projects.

A heads-up: custom skills you download from the internet run as instructions inside Ayre. Only use skills from sources you trust.

Handoff built-in

Writes a session summary and saves it to your active project folder. Use the Handoff button in the top bar at the end of a session to preserve context for next time.

Skills work best when built iteratively — start with a rough draft, then have a back-and-forth with Ayre in chat to refine it. Test each part of the workflow before moving on.

Settings

Defaults, hardware tuning, and appearance.

Connection

The local port Ayre's interface serves on. Pick any 4-digit port (1000–9999) if 2500 is taken. Loopback only — never exposed off this machine. Takes effect after you restart Ayre.

Session

When you press Handoff →, Ayre asks the model to write a session summary and save it to your project folder. Some reasoning models call the save tool repeatedly in a loop. The cooldown blocks duplicate saves within this window — the model still gets a success response so it stops trying. Set higher if your model keeps producing extra files; set lower if you do back-to-back handoffs in the same session.

min

Memory

Ayre's persistent memory holds facts it has saved about you and your work — your preferences, role, and ongoing project context. Memory is user-level: shared across all projects and loaded into every session automatically. Clearing permanently deletes those saved notes — it can't be undone. It does not turn memory off, so Ayre can save fresh notes again afterwards. Turn memory on or off from the Workspace tab.

Appearance

Six runtime-swappable themes — every color, font and effect is a CSS variable. Use the switcher in the top bar; the signature gunmetal/orange/cyan palette ships as Ayre and is the default.

Reduce motion quiets Ayre's ambient animations — the thinking-visual tendrils, the status-dot pulse, the streaming cursor, the working indicator, and the tip ticker's slide. System follows your operating system's reduced-motion setting; On and Off force it either way. The tip ticker keeps rotating tips regardless — it just swaps them in place instead of sliding when motion is reduced.

REDUCE MOTION

The Ayre thinking visual — a tendril field that blooms from the center of the chat, behind your messages, while the model is generating (more tendrils reaching further the longer it thinks), then fades out. Purely ambient; it animates only during a reply. The context meter's two bars are always shown either way. Suppressed while Reduce Motion (above) is active — the two are independent gates.
